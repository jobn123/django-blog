# django-blog
django-blog 

--------

## install 

```shell
  git clone https://github.com/jobn123/django-blog
```

## Usage
```
cd django-blog
pip install mysql-python
mysql -u root -p
create database xblog default charset utf8 collate utf8_general_ci;
python manage.py migrate
python manage.py runserver
```
